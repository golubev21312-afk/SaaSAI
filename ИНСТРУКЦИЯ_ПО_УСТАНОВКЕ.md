# 🤖 Инструкция по установке расширения
## "ИИ Ответы для GetCourse" — Клиника Картавенко

---

## ШАГ 1 — Распакуйте архив

1. Скачайте файл **getcourse-ai-extension.zip**
2. Щёлкните правой кнопкой мыши → **"Извлечь всё"**
3. Запомните папку куда распаковали (например: Рабочий стол → getcourse-ai-extension)

---

## ШАГ 2 — Установите расширение в Chrome

1. Откройте браузер **Google Chrome**
2. В адресной строке введите: `chrome://extensions`
3. Нажмите Enter
4. В правом верхнем углу включите переключатель **"Режим разработчика"**
5. Нажмите кнопку **"Загрузить распакованное"**
6. Выберите папку **getcourse-ai-extension** (которую распаковали в Шаге 1)
7. Расширение появится в списке ✅

---

## ШАГ 3 — Настройте расширение

1. В правом верхнем углу Chrome нажмите на иконку 🧩 (Расширения)
2. Найдите **"ИИ Ответы для GetCourse"** и нажмите на иконку
3. Откроется окно настроек:
   - **API Ключ Groq** → вставьте ключ (начинается с `gsk_...`)
   - **Имя куратора** → введите своё имя (например: Марина Юрьевна)
4. Нажмите **"Сохранить настройки"**

---

## ШАГ 4 — Используйте!

1. Откройте **GetCourse** → страницу с ответом ученика
2. В правом нижнем углу появится синяя кнопка **🤖 Составить ответ**
3. Нажмите кнопку
4. ИИ прочитает текст ученика и сгенерирует персональный ответ (5-10 секунд)
5. Прочитайте ответ, при необходимости отредактируйте прямо в окне
6. Нажмите **✅ Вставить в поле** — текст появится в поле для ответа
7. Отправьте как обычно!

---

## ❓ Частые вопросы

**Кнопка не появляется?**
→ Убедитесь что вы на странице `teach/control/answers` в GetCourse
→ Обновите страницу (F5)

**Ошибка "API ключ не настроен"?**
→ Откройте настройки расширения и сохраните ключ

**Ответ не вставляется в поле?**
→ Скопируйте текст из окна расширения вручную (Ctrl+A, Ctrl+C)

**Нужен новый API ключ Groq?**
→ Зайдите на console.groq.com → API Keys → Create API Key (бесплатно!)
