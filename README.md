# ИИ Ответы для GetCourse — Клиника Картавенко

Расширение для Chrome, которое автоматически генерирует персональные ответы кураторов на отчёты учеников на платформе GetCourse. Работает на базе языковой модели Llama 3.3 через API Groq.

---

## Возможности

### Генерация ответов
- Генерирует персональный ответ под каждый отчёт ученика одним кликом
- Три варианта **длины**: Короткий (40–80 слов) / Средний (100–200) / Подробный (250–400)
- Три варианта **тона**: Тёплый / Официальный / Кратко
- Автоматически определяет настроение ученика (позитивный / жалоба / нейтральный) и адаптирует ответ

### Быстрые правки
- **Короче** — сокращает готовый ответ вдвое
- **Теплее** — делает текст более эмпатичным
- **Перефразировать** — другими словами, тот же смысл

### Удобство работы
- **Горячая клавиша** `Alt+G` — генерирует ответ без клика мышью
- **Пакетная генерация** — кнопка "Сгенерировать все" обрабатывает все ответы на странице последовательно
- **Вставка в поле** — готовый текст вставляется прямо в textarea GetCourse одним кликом
- **Копировать** — копирует текст в буфер обмена
- **Сохранение черновиков** — при обновлении страницы сгенерированный текст восстанавливается
- **Счётчик слов** — отображается в шапке панели

### Шаблоны
Сохраняйте часто используемые фразы и вставляйте их в ответ одним кликом через кнопку "Шаблоны ▾". Управление шаблонами — в настройках расширения.

### Статистика
В попапе расширения отображается количество сгенерированных ответов за сегодня и за неделю.

### Настройки
- Редактор системного промпта — можно полностью переписать инструкцию для ИИ
- Тумблеры для каждой функции: индикатор настроения, автосвёртка, пакетная кнопка, черновики

## Установка

### 1. Получите API ключ Groq
1. Зарегистрируйтесь на [console.groq.com](https://console.groq.com)
2. Перейдите в раздел **API Keys**
3. Нажмите **Create API Key** и скопируйте ключ

### 2. Установите расширение в Chrome
1. Откройте `chrome://extensions/`
2. Включите **Режим разработчика** (переключатель в правом верхнем углу)
3. Нажмите **Загрузить распакованное расширение**
4. Выберите папку с файлами расширения

### 3. Настройте расширение
1. Кликните на иконку расширения в панели Chrome
2. Введите API ключ Groq
3. Введите имя куратора (будет подставляться в подпись каждого ответа)
4. Нажмите **Сохранить настройки**

---

## Использование

1. Перейдите на страницу ответов учеников:
   `https://pro.klinikakartavenko.com/teach/control/answers/...`
2. Под каждым ответом ученика появится индикатор настроения и кнопка **Составить ответ ИИ**
3. Выберите длину и тон ответа
4. Нажмите кнопку или используйте `Alt+G`
5. Отредактируйте при необходимости
6. Нажмите **Вставить в поле** — текст появится в поле комментария GetCourse
